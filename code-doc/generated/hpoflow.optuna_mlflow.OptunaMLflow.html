hpoflow.optuna_mlflow.OptunaMLflow¶

class hpoflow.optuna_mlflow.OptunaMLflow(tracking_uri=None, num_name_digits=3, enforce_clean_git=False, optuna_result_name='optuna_result')¶

Wrapper to log to Optuna and MLflow at the same time.

__init__(tracking_uri=None, num_name_digits=3, enforce_clean_git=False, optuna_result_name='optuna_result')¶

Init class.

Parameters

  • tracking_uri ([str], optional) – See MLflow documentation. Defaults to None which logs to the default locale folder ./mlruns or uses the MLFLOW_TRACKING_URI environment variable if it is available.

  • num_name_digits (int, optional) – [description]. Defaults to 3.

  • enforce_clean_git (bool, optional) – Check and enforce that the GIT repository has no uncommited changes. Defaults to False. Also see git.repo.base.Repo.is_dirty
